select选择器使用文档

  • 使用方法：
    1、在当前模块的.module.ts中引入:
    import { SharedModule } from '../shared/shared.module';
  •     添加到@NgModule装饰器中:
    @NgModule({
        imports: [
            SharedModule
        ]
    });
  • 2、.component.ts中,声明组件需要的数据，如:
    selects1: Array<any> = [];
    ngOnInit() {
        this.selects1 = [
            { text: '一年级', value: '01' },
            { text: '二年级', value: '02, disabled: true },
            { text: '三年级', value: '03' },
            { text: '四年级', value: '04' },
            { text: '五年级', value: '05' },
            { text: '六年级', value: '06' }
        ];
    };
  •     定义接收组件内部向外传递事件方法
    selected1(event: any) {
        this.selectValue1 = event;
    };
  •     当配置清除功能时，定义接收组件内部向外传递的清除事件方法
    clear() {
        this.selectValue1 = null;
    };
  • 3、.component.html中，直接引用组件标签，传入需要的配置字段，如：
  • <app-select-selection [label]="'年级'" [selects]="selects4" (selected)="selected1($event)" [defaultValue]="defaultValue" [clear]="true" (clearEventer)="clear()"></app-select-selection>
  • 4、组件参数说明：
    参数 说明 类型 可选值 默认值
    selects 必选，数据源 Array<{text:string,value:string,disabled?:Boolean}> — -
    label 必选，label标签名 String — -
    showType 可选，显示类型（inline,block） string inline,block inline
    selected 选择时向外传递事件方法，同时向外传递选择值 — — -
    defaultValue 可选，为组件内部传递默认值 string — -
    clearEventer 可选，配置清除功能时可用，点击清除时向外传递事件 — — -
    disabled 当item项禁选时，数据对象添加‘disabled：true’字段即可 — — -
