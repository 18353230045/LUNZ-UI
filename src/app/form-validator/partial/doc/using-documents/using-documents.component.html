官方使用文档见： “ng2-validation”. 此表单验证器，包括常见的必填验证、数字验证、最小/大值验证、字符长度验证、日期验证、最小/大日期验证、 email验证、url验证、字段相同验证等，电话号码验证不能针对国内常见电话号码格式验证， 因此在demo中利用正则表达式来验证。各种验证方法雷同，这里拿required举例，更多不常见信息验证请查看 官网。

  • 1、在当前模块的.module.ts中引入:
    import { FormsModule } from '@angular/forms';
    import { CustomFormsModule } from 'ng2-validation';
  •     添加到@NgModule装饰器中:
    @NgModule({imports: [FormsModule,CustomFormsModule]});
  • 2、.component.ts中:
    import { FormGroup,FormControl } from '@angular/forms';
    import { CustomValidators } from 'ng2-validation';
  •     class类中,定义form变量实现接口FormGroup，构造函数中定义验证规则，详见demo,如：required,‘required-validator.component.ts’
  • 3、.component.html中:
  •     在需要验证的表单控件input中声明模板引用变量，值等于ngModel,并添加验证规则为‘required’，如：
    <input type="text" class="form-control" [(ngModel)]="form.required" name="required" #field="ngModel" required />
        添加验证视觉提示，根据模板引用变量中验证信息来显示与隐藏，如：
    <p *ngIf="field.errors?.required" class="text-danger">此字段为必填字段</p>
  •     详细见demo文件，‘required.component.html’
