表单验证器使用文档

官方使用文档见： “ng2-validation”. 此表单验证器，包括常见的必填验证、数字验证、最小/大值验证、字符长度验证、日期验证、最小/大日期验证、 email验证、url验证、字段相同验证等，电话号码验证不能针对国内常见电话号码格式验证， 因此在demo中利用正则表达式来验证。各种验证方法雷同，这里拿required举例，更多不常见信息验证请查看 官网。

  • 1、在当前模块的.module.ts中引入:
    import { FormsModule,ReactiveFormsModule } from '@angular/forms';
    import { SharedModule } from '../shared/shared.module';
  •     添加到@NgModule装饰器中:
    @NgModule({imports: [SharedModule,FormsModule,ReactiveFormsModule]});
  • 2、.component.ts中:
    import { FormBuilder, FormControl, FormGroup, Validators } from '@angular/forms';
    import { CustomValidators } from 'ng2-validation';
  •     class类中,定义form变量实现接口FormGroup，构造函数中定义验证规则，详见demo,如：'form-validator.component.ts’
  • 3、.component.html中，以验证email为例：
  • <form [formGroup]="form">
        <input type="text" class="form-control" name="email" formControlName="email" />
        <app-validation-message [control]="form.controls['email']"></app-validation-message>
    </form>
  •     注：ng2-validation验证器中的phone验证电话号码不能针对国内电话号码规则正确验证，使用angular正则表达式来进行验证。     详细见demo文件，‘form-validator.component.html’
