文件上传使用文档

  • 使用方法：
    1、在当前模块的.module.ts中引入:
    import { SharedModule } from '../shared/shared.module';
  •     添加到@NgModule装饰器中:
    @NgModule({
        imports: [
            SharedModule
        ]
    });
  • 2、.component.html中,直接引用上传组件标签，传入配置，如：
    <app-file-upload [multiple]="true" [accept]="'image/*'" (success)="successHandle($event)" (error)="errorHandle($event)"> </app-file-upload>
  • 3、.component.ts中定义接收上传和失败时传出事件，详细见demo文件'file-upload-demo.component.ts'
  • 4、组件参数说明：
    参数 说明 类型 可选值 默认值
    {{params.name}} {{params.explain}} {{params.type}} {{params.optionaValue}} {{params.defaultValue}}
