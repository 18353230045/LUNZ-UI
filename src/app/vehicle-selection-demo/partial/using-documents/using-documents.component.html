车型选择使用方法

使用方法：
  • 1、在当前模块的.module.ts中引入车型选择组件和组件所需要的服务，如下:
    import { SharedModule } from '../shared/shared.module';
  •     添加到@NgModule装饰器中:
    @NgModule({
        imports: [
             SharedModule
        ]
    });
  • 2、.component.html中:
  •     在使用车型选择组件的视图（.component.html）中，直接引用组件标签即可， 传入需要组件传出的值的类型[‘brand’：车品牌（id/name），‘series’：车品牌(id/name)+车系列(id/name)， ‘model’：车品牌(id/name)+车系列(id/name)+车型所有信息]和接受组件传出的事件方法，此两项必传。例如：
    <app-vehicle-selection [outputType]="'brand'" (selected)="getBrand($event)"></app-vehicle-selection>
  • 3、.component.ts中:
    在.component.ts只需声明用于接受组件传出的事件方法，例如：
    getBrand(event: any) {
        somecode(event);
    };
    event即是组件传出的数据，详细见demo
  • 4、组件参数说明：
    参数 说明 类型 可选值 默认值
    {{params.name}} {{params.explain}} {{params.type}} {{params.optionaValue}} {{params.defaultValue}}
