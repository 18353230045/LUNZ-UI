使用文档

  • 使用方法：
    1、在当前模块的.module.ts中引入:
    import { SharedModule } from '../shared/shared.module';
  •     添加到@NgModule装饰器中:
    @NgModule({
        imports: [
            SharedModule
        ]
    });
  • 2、.component.html中，直接引用组件标签，如：
  • <app-datepicker [id]="'default'" (selected)="getValue($event)"></app-datepicker>
  • 3、.component.ts中声明用于接受组件传出的事件方法，如：
    export class DatepickerSelectComponent implements OnInit {
        date:String;

        constructor() { };

        ngOnInit() { };

        getDate(event: any) {
            this.date = event;
        };
    };
  • 4、组件参数说明：
    参数 说明 类型 可选值 默认值
    {{params.name}} {{params.explain}} {{params.type}} {{params.optionaValue}} {{params.defaultValue}}
