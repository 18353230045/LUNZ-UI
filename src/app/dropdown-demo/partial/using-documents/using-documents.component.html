使用文档

  • 使用方法：
    1、在当前模块的.module.ts中引入:
    import { SharedModule } from '../shared/shared.module';
  •     添加到@NgModule装饰器中:
    @NgModule({
        imports: [
            SharedModule
        ]
    });
  • 2、.component.html中，直接引用组件标签，如：
  • <app-dropdown-selection [options]="options" [position]="'left'" [dropText]="'部门'" [class]="'btn-secondary'" (selected)="handle1($event)"></app-dropdown-selection>
  • 3、.component.ts中声明传入组件的数据源变量，如：
    export class DropdownComponent implements OnInit {
        options: Array<any> = [];

        constructor() { };

        ngOnInit() {
            this.options = [
                { text: 'item1', id: '01' },
                { text: 'item2', id: '02', disabled: true },
                { text: 'item3', id: '03' },
                { text: 'item4', id: '04' },
                { text: 'item5', id: '05' },
                { text: 'item6', id: '06' }
            ];
        };
    };
  • 4、组件参数说明：
    参数 说明 类型 可选值 默认值
    {{params.name}} {{params.explain}} {{params.type}} {{params.optionaValue}} {{params.defaultValue}}
