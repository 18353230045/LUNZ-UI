使用文档

  • 使用方法：
    1、在当前模块的.module.ts中引入:
    import { SharedModule } from '../shared/shared.module';
  •     添加到@NgModule装饰器中:
    @NgModule({
        imports: [
            SharedModule
        ]
    });
  • 2、.component.ts中声明传入组件的数据源变量，如：
    export class DropdownComponent implements OnInit {
        options: Array<any> = [];
        constructor() { };
        ngOnInit() {
            this.options = [
                { text: 'item1', id: '01' },
                { text: 'item2', id: '02', disabled: true },
                { text: 'item3', id: '03' },
                { text: 'item4', id: '04' },
                { text: 'item5', id: '05' },
                { text: 'item6', id: '06' }
            ];
        };
    };
  • 3、.component.html中，直接引用组件标签，如：
  • <app-dropdown-selection [options]="options" [position]="'left'" [dropText]="'部门'" [class]="'btn-secondary'" (selected)="handle1($event)"></app-dropdown-selection>
  • 4、组件参数说明：
    参数 说明 类型 可选值 默认值
    options 必选，数据源 Array<{text:string,id:string,disabled?:Boolean}> — -
    type 可选，组件类型（btn,icon） string btn,icon btn
    position 可选，组件展开位置 string center,right,left center
    dropText 可选，为btn型时可用，显示列表title，为icon型时不可用 string — -
    class 可选，为dropdown触发按钮添加样式 string — -
    iconClass 可选，为icon型可用，为dropdown触发按钮添加图标 string — la la-ellipsis-h
    itemIcon 可选，为item添加图标 string — flaticon-share
    trigger 可选，选择触发dropdown方式 string click,hover click
    multiselect 可选，btn型可用，选择是否支持多选 Boolean true,false false
    selected 选择item项时向外传递事件方法，同时向外传递选择的item值 — — -
