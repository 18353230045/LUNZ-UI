进度条使用文档

  • 说明：
    进度条组件使用了第三方组件 'element-angular'。
  • 使用方法：
    1、在当前模块的.module.ts中引入:
    import { ElModule } from 'element-angular';
  •     添加到@NgModule装饰器中:
    @NgModule({
        imports: [
            ElModule.forRoot(),
        ]
    });
  • 2、.component.html中,直接使用组件标签：
  • <el-progress [text-inside]="true" [stroke-width]="14" [percentage]="0"></el-progress>
  • 3、组件参数说明：
    参数 说明 类型 可选值 默认值
    {{params.name}} {{params.explain}} {{params.type}} {{params.optionaValue}} {{params.defaultValue}}
